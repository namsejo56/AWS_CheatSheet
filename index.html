EC2 (Elastic Compute Cloud)

Virtual servers in the cloud

  • Instance types: t3, m5, c5, r5, etc.
  • On-Demand, Reserved, Spot instances
  • Auto Scaling Groups
  • Load Balancers (ALB, NLB, CLB)

Lambda

Serverless compute service

  • Event-driven execution
  • 15-minute max execution time
  • Supports multiple runtimes
  • Pay per invocation

ECS/EKS

Container orchestration

  • ECS: AWS managed containers
  • EKS: Managed Kubernetes
  • Fargate: Serverless containers
  • ECR: Container registry

üíæ Storage Services

S3 (Simple Storage Service)

Object storage service

  • Storage classes: Standard, IA, Glacier
  • Bucket policies and ACLs
  • Versioning and lifecycle policies
  • Cross-region replication

EBS (Elastic Block Store)

Block storage for EC2

  • Volume types: gp2, gp3, io1, io2
  • Snapshots and encryption
  • Multi-Attach capability
  • Elastic Volumes

EFS (Elastic File System)

Managed NFS file system

  • POSIX-compliant
  • Scalable performance
  • Multiple availability zones
  • Backup and encryption

RDS (Relational Database Service)

Managed relational databases

  • MySQL, PostgreSQL, Oracle, SQL Server
  • Multi-AZ deployments
  • Read replicas
  • Automated backups

DynamoDB

NoSQL database service

  • Key-value and document database
  • Serverless and provisioned modes
  • Global tables
  • DynamoDB Streams

ElastiCache

In-memory caching

  • Redis and Memcached
  • Cluster mode
  • Backup and restore
  • Multi-AZ failover

  • Private hosted zones
